u8  SlotNum,
XAie_Packet  Pkt,
u8  Mask,

u8  Arbitor 
)

This API is used to configure the stream switch slave slot configuration registers.

This API should be used in combination with other APIs to first configure the master and slave ports in packet switch mode.

Parameters
DevInst,:Device Instance
Loc,:Loc of AIE Tiles
Slave,:Slave port type
SlvPortNum,:Slave port number
SlotNum,:Slot number for the slave port
Pkt,:Packet with initialized packet id and packet type
Mask,:Mask field in the slot register
MSel,:Msel register field in the slave slot register
Arbitor,:Arbitor to use for this packet switch connection
Returns
XAIE_OK on success, Error code on failure.
Note
None.
