Data Structures
Here are the data structures with brief descriptions:
aie_dma_bd_argsStruct aie_dma_bd - AIE DMA buffer descriptor information : DMA buffer descriptor : virtual address of the data : Tile location relative to the start of a partition : buffer descriptor id
aie_dmabuf_bd_argsStruct aie_dmabuf_bd_args - AIE dmabuf buffer descriptor information : DMA buffer descriptor, within the buffer descriptor, the address field will be the offset to the start of the dmabuf : DMA buffer handler which is dmabuf file descriptor : Tile location relative to the start of a partition : buffer descriptor id
aie_locationStruct aie_location - AIE location information : column id : row id
aie_memStruct aie_mem - AIE memory information : range of tiles of the memory : register offset within a tile of the memory : of a the memory in one tile : file descriptor of the memory
aie_mem_argsStruct aie_mem_args - AIE memory enquiry arguments : number of "struct aie_mem" elements e.g
aie_partition_queryStruct aie_partition_query - AIE partition query arguments : number of defined partitions in the system : buffer to store defined partitions information
aie_partition_reqStruct aie_partition_req - AIE request partition arguments : partition node id
aie_rangeStruct aie_range - AIE range information : start tile location : size of the range, number of columns and rows
aie_range_argsStruct aie_range_args - AIE range request arguments : partition id
aie_reg_argsStruct aie_reg_args - AIE access register arguments : if this request is to read, write or poll register : mask for mask write, 0 for not mask write : offset of register to the start of an AI engine partition : value to write or get
aie_tiles_arrayStruct aie_tiles_array - AIE tiles array : tiles locations array : number of tiles in the tiles locations array
ion_allocation_dataDOC: Ion Userspace API
ion_heap_queryStruct ion_heap_query - collection of data about all heaps - total number of heaps to be copied - buffer to copy heap data
