This API enables clock for all the tiles passed as argument to this API. 


Parameters

  

    
DevInst,:Device Instance 

    
Loc,:Location of AIE tile 

    
NumTiles,:Number of tiles requested for use.

  

  




Returns
XAIE_OK on success.


Note
As all the tiles are gated when the system boots, this function needs to be called after device instance is initialized and before any other AI engine operations. Otherwise, the other AI engine functions may access gated tiles. 



References _XAie_GetTileBitPosFromLoc(), and _XAie_SetBitInBitmap().
