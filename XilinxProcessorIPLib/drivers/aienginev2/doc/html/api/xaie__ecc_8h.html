void _XAie_EccEvntResetPM ( XAie_DevInst *  DevInst,
XAie_LocType  Loc 
)

This API configures the register to turn ECC Off for Program memory of the given tile.

Parameters
DevInst,:Device Instance
Loc,:Location of tile
Returns
none
Note
This API is internal, hence all the argument checks are taken care of in the caller API. This API resets the Ecc scrubbing event register for program memory of the given tile. It does not release performance counter 0.

AieRC _XAie_EccOnDM ( XAie_DevInst *  DevInst,
XAie_LocType  Loc 
)

This API configures registers to turn ECC On for Data memory of the given tile.

Parameters
DevInst,:Device Instance
Loc,:Location of AIE tile
Returns
none
Note
This API is internal, hence all the argument checks are taken care of in the caller API. To turn ECC On for Data Memory of a given tile, the performance counter 0 event is broadcasted from core module to memory module on broadcast channel 6 to trigger ECC ON for data memory.ECC scrubbing event register of mem module is configured with that generated event.

References _XAie_GetTileBitPosFromLoc(), _XAie_GetTileTypefromLoc(), _XAie_SetBitInBitmap(), XAie_EventBroadcast(), and XAie_EventBroadcastBlockDir().

This API configures registers to turn ECC On for Program memory of the given tile.

Parameters
DevInst,:Device Instance
Loc,:Location of tile
Returns
none
Note
This API is internal, hence all the argument checks are taken care of in the caller API. To turn ECC On for program memory of a given tile, the performance counter 0 event is generated every 10^6 clock cycle to trigger ECC ON for program memory. ECC scrubbing event register of core module is configured with that generated event.
