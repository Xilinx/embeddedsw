AieRC XAie_LoadElf ( XAie_DevInst *  DevInst,
XAie_LocType  Loc,
const char *  ElfPtr,
u8  LoadSym 
)

This function loads the elf from file to the AIE Cores.

The function writes 0 for the unitialized data section.

Parameters
DevInst,:Device Instance.
Loc,:Location of AIE Tile.
ElfPtr,:Path to the elf file.
LoadSym,:Load symbols from .map file. This argument is valid when AIESIM is defined.
Returns
XAIE_OK on success and error code for failure.
Note
None.

References _XAie_GetTileTypefromLoc(), and XAie_LoadElfMem().
