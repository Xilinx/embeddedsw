The upsizer register is configured with the PortNumber provided.

AieRC XAie_PlToAieIntfDisable ( XAie_DevInst *  DevInst,
XAie_LocType  Loc,
u8  PortNum,
XAie_PlIfWidth  Width 
)

This API Disables the stream port with width for PL->AIE interfaces.

The downsizer register is configured with the port number provided by the user. Once the downsizer register is configured, the API also enables the ports in the downsizer enable register.

Parameters
DevInst,:Device Instance
Loc,:Loc of AIE Tiles
PortNum,:Stream Port Number (0-7)
Width,:Supported widths are 32, 64 and 128 (PLIF_WIDTH_32/64/128)
Returns
XAIE_OK on success, Error code on failure.
Note
If this API is used to configure PLTOAIE interfaces, explicit call to disable stream ports in downsizer enable register is not required.
