This API reset the SHIM for the specified columns.

Parameters
DevInst,:Device Instance
StartCol,:Start column
NumCols,:Number of columns
Returns
XAIE_OK
Note
It is not required to check the DevInst as the caller function should provide the correct value. Also it will not check the nor the , the caller function should do the validation and provide the correct values. This function does the following steps:
  • Set reset bit for every SHIM
  • Assert SHIM reset
  • Deassert SHIM reset
  • Unset reset bit for every SHIM
