AieRC XAie_DataMemBlockRead ( XAie_DevInst *  DevInst,
XAie_LocType  Loc,
u32  Addr,
void *  Dst,
u32  Size 
)

This API reads a block of data from the specified data memory location of the selected tile.

Byte-level reads are supported by this API. For unaligned data memory offsets, this API implements read-modify-write operation.

Parameters
DevInst,:Device Instance
Loc,:Loc of AIE Tiles
Addr,:Address in data memory to write.
Dst- Destination to store read data.
Size- Size in bytes to read.
Returns
XAIE_OK on success and error code on failure
Note
None.
