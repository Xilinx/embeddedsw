AieRC XAie_TraceControlConfig ( XAie_DevInst *  DevInst,
XAie_LocType  Loc,
XAie_ModuleType  Module,
XAie_Events  StartEvent,
XAie_Events  StopEvent,
XAie_TraceMode  Mode 
)

This API is used to configure trace control register with event number to start tracing, stop tracing and the mode for valid modules.

Parameters
DevInst,:Device Instance
Loc,:Location of AIE Tile
Module,:Module of tile. For AIE Tile - XAIE_MEM_MOD or XAIE_CORE_MOD, For Shim tile - XAIE_PL_MOD,
StartEvent,:Event number to start tracing.
StopEvent,:Event number to stop tracing.
Mode,:Trace module can operate in the following modes, EVENT_TIME: Valid for all module types EVENT_PC, INST_EXEC: Only valid for core module.
Returns
XAIE_OK on success, error code on failure.
Note
None.

References _XAie_CheckModule(), and _XAie_GetTileTypefromLoc().
