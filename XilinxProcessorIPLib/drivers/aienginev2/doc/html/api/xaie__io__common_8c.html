This API grants resource based on availibility for the given location and marks that rsc as in use in the relevant bitmap. 


The API checks for contiguous bits available in the bitmap.


Parameters

  

    
Bitmap,:Bitmap of the resource 

    
StartBit,:Index for the resource start bit in the bitmap 

    
StaticBitmapOffset,:Offset for static bitmap 

    
NumRscPerTile,:Number of resource requested per tile 

    
MaxRscVal,:Maximum number of resource per tile 

    
RscArrPerTile,:Pointer to store available resource 

    
NumContigRscs,:Number of conitguous resource required

  

  




Returns
XAIE_OK on success.

Internal only.
