XAie_LocType  Loc,
const XAie_DmaMod *  DmaMod,
u8  ChNum,
XAie_DmaDirection  Dir,
u8 *  PendingBd 
)

This API is used to get the count of scheduled BDs in pending.

Parameters
DevInst,:Device Instance
Loc,:Location of AIE Tile
DmaMod,:Dma module pointer
ChNum,:Channel number of the DMA.
Dir,:Direction of the DMA Channel. (MM2S or S2MM)
PendingBd,:Pointer to store the number of pending BDs.
Returns
XAIE_OK on success, Error code on failure.
Note
Internal only. For AIE Tiles only.
AieRC _XAie_DmaSetInterleaveEnable ( XAie_DmaDesc *  DmaDesc,
u8  DoubleBuff,
u8  IntrleaveCount,
u16  IntrleaveCurr 
)

This API setups DmaDesc with parameters to run dma in interleave mode for AIE.

Parameters
DmaDesc,:Initialized Dma Descriptor.
DoubleBuff,:Double buffer to use(0 - A, 1-B)
IntrleaveCount,:Interleaved count to use(to be 32b word aligned)
IntrleaveCurr,:Interleave current pointer.
Returns
XAIE_OK on success, Error code on failure.
Note
Internal only. The API sets up the value in the dma descriptor and does not configure the buffer descriptor field in the hardware.
AieRC _XAie_DmaSetLock ( XAie_DmaDesc *  DmaDesc,
XAie_Lock  Acq,
XAie_Lock  Rel,
u8  AcqEn,
u8  RelEn 
)

This API initializes the Acquire and Release Locks for a a Dma for AIE descriptor.

Parameters
DmaDesc,:Initialized Dma Descriptor.
Acq,:Lock object with acquire lock ID and lock value.
Rel,:Lock object with release lock ID and lock value.
AcqEn,:XAIE_ENABLE/XAIE_DISABLE for enable or disable acquire lock.
RelEn,:XAIE_ENABLE/XAIE_DISABLE for enable or disable release lock.
Returns
XAIE_OK on success, Error code on failure.
Note
Internal Only. Should not be called directly. This function is called from the internal Dma Module data structure.
