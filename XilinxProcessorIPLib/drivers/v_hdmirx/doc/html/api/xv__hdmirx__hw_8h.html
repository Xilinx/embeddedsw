xv_hdmirx_hw.h File Reference

Overview

This header file contains identifiers and register-level core functions (or macros) that can be used to access the Xilinx HDMI RX core.

For more information about the operation of this core see the hardware specification and documentation in the higher level driver xv_hdmirx.h file.
