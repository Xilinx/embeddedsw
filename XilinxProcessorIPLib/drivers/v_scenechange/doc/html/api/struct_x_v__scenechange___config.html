Detailed Description

Configuration structure for the Scene Change Detection IP core.

This structure contains the configuration parameters for the Xilinx Scene Change Detection video processing IP. It includes device identification, memory addressing, format support settings, and resource constraints.

Note
The structure uses conditional compilation (SDT - Scoped Device Tree) to support both legacy and device tree based configurations.
