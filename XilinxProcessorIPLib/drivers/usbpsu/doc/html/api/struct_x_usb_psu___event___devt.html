struct XUsbPsu_event_devt - Device Events : indicates this is a non-endpoint event : indicates it's a device event.

Should read as 0x00 : indicates the type of device event. 0 - DisconnEvt 1 - USBRst 2 - ConnectDone 3 - ULStChng 4 - WkUpEvt 5 - Reserved 6 - EOPF 7 - SOF 8 - Reserved 9 - ErrticErr 10 - CmdCmplt 11 - EvntOverflow 12 - VndrDevTstRcved : Reserved, not used : Information about this event : Reserved, not used
