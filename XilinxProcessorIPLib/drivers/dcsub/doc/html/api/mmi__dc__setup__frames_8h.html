void XDpDc_CreateDescriptors ( RunConfig *  RunCfgPtr,
XDcDma_Descriptor *  XDesc,
FrameInfo *  FBInfo,
XDcDma_Descriptor *  NextDesc 
)

This function initializes a DMA descriptor with source address, data size, line size, stride, and next descriptor linkage from the given FrameInfo.

For cursor descriptors, Target_Addr is also set.

Parameters
RunCfgPtris a pointer to the application configuration structure.
XDescis a pointer to the DMA descriptor to initialize.
FBInfois a pointer to the FrameInfo containing buffer attributes.
NextDescis a pointer to the next descriptor in the chain, 
    or NULL for a self-linked (last) descriptor.
Returns
None.
Note
None.

void XDpDc_InitFrames ( RunConfig *  RunCfgPtr)

This function initializes video, cursor, and audio frame buffers.

Buffer initialization is format-aware: the number of planes per stream is

determined by the video format mode (Interleaved/SemiPlanar/Planar). V1-V3 are Stream1 planes, V4-V6 are Stream2 planes.

Parameters
RunCfgPtris a pointer to the application configuration structure.
Returns
None.
Note
Descriptor creation is handled separately by XDpDc_SetupStream1Descriptors and XDpDc_SetupStream2Descriptors, which must be called after DMA hardware initialization.

void XDpDc_SetupStream1Descriptors ( RunConfig *  RunCfgPtr)

This function creates DMA descriptors for Stream1 and assigns them to Video DMA channels 0-2.

The number of descriptors depends on the video format mode: Interleaved=1, SemiPlanar=2, Planar=3. Uses switch fall-through from Planar -> SemiPlanar -> Interleaved.

Parameters
RunCfgPtris a pointer to the application configuration structure.
Returns
None.
Note
Must be called after XDcDma_CfgInitialize().
