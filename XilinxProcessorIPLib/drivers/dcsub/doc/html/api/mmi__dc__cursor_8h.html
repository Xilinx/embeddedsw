This function configures cursor blend parameters using DCsub APIs.

Parameters

Returns
None
Note
Uses XDcSub_SetCursorBlend and XDcSub_SetSdpCursorBuffers APIs to configure cursor parameters. Must be called BEFORE hardware init.

void XDpDc_FillCursorBuffer ( RunConfig *  RunCfgPtr)

This function fills the cursor buffer with a red circle pattern.

Parameters
RunCfgPtr- Pointer to RunConfig structure
Returns
None
Note
RGBA4444 format: 16-bit per pixel, bit layout from MSB to LSB: Bits 15-12: Alpha[3:0] Bits 11-8: Blue[3:0] Bits 7-4: Green[3:0] Bits 3-0: Red[3:0]
      Red (opaque) = 0xF00F (A=F, B=0, G=0, R=F)
      Transparent = 0x0000 (all zeros)
