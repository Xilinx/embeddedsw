int XV_axi4s_remap_Initialize ( XV_axi4s_remap *  InstancePtr,
UINTPTR  BaseAddress 
)

Initialize AXI4-Stream Remap driver instance by base address (SDT)

This function initializes an AXI4-Stream Remap driver instance by looking up the device configuration based on the physical base address, then calling the configuration initialization function with the retrieved configuration.

Parameters
InstancePtrPointer to the driver instance structure to initialize
BaseAddressPhysical base address of the hardware instance
Returns
XST_SUCCESS if initialization completed successfully XST_DEVICE_NOT_FOUND if base address not found in configuration table Other error codes from XV_axi4s_remap_CfgInitialize()
Note
This is the SDT (System Device Tree) version that uses base addresses instead of device IDs
InstancePtr->IsReady is set to 0 if device lookup fails

References XV_axi4s_remap_Config::BaseAddress, XV_axi4s_remap::IsReady, XV_axi4s_remap_CfgInitialize(), and XV_axi4s_remap_LookupConfig().

XV_axi4s_remap_Config* XV_axi4s_remap_LookupConfig ( UINTPTR  BaseAddress)

Lookup configuration structure by base address (SDT)

This function searches the configuration table for an AXI4-Stream Remap device matching the specified base address and returns a pointer to its configuration structure. If BaseAddress is 0, returns the first entry.

Parameters
BaseAddressPhysical base address of the device to search for, or 0 to return the first available device
Returns
Pointer to matching XV_axi4s_remap_Config structure if found NULL if no matching base address is found
Note
This is the SDT (System Device Tree) version that uses base addresses instead of device IDs
