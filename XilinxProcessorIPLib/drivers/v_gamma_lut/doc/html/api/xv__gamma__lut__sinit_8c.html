int XV_gamma_lut_Initialize ( XV_gamma_lut *  InstancePtr,
UINTPTR  BaseAddress 
)

Initializes the Gamma LUT driver instance (SDT)

This function initializes a Gamma LUT driver instance by looking up the configuration based on the base address and calling the configuration initialization function. This version is used with System Device Tree (SDT) configurations.

Parameters
InstancePtris a pointer to the XV_gamma_lut instance.
BaseAddressis the base address to initialize.
Returns
XST_SUCCESS on success, XST_DEVICE_NOT_FOUND if device not found
Note
None

References XV_gamma_lut_Config::BaseAddress, XV_gamma_lut::IsReady, XV_gamma_lut_CfgInitialize(), and XV_gamma_lut_LookupConfig().

XV_gamma_lut_Config* XV_gamma_lut_LookupConfig ( UINTPTR  BaseAddress)

Looks up the device configuration based on base address (SDT)

This function searches the configuration table for a device matching the provided base address and returns a pointer to its configuration structure. This version is used with System Device Tree (SDT) configurations.

Parameters
BaseAddressis the base address to search for.
Returns
Pointer to the XV_gamma_lut_Config structure if found, NULL otherwise

None
