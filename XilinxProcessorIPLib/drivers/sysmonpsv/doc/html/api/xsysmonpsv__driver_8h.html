Typedef Documentation

typedef void(* XSysMonPsv_Handler)(void *CallbackRef)

This data type defines a handler that an application defines to communicate with interrupt system to retrieve state information about an application.

Parameters
CallbackRefCallback reference passed in by the upper layer when setting the handler, and is passed back to the upper layer when the handler is called. It is used to find the device driver instance.
