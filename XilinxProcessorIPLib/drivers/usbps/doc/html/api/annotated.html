Data Structures
Here are the data structures with brief descriptions:
XUsbPsThe XUsbPs driver instance data
XUsbPs_ConfigThe XUsbPs_Config structure contains configuration information for the USB controller
XUsbPs_DeviceConfigThe XUsbPs_DeviceConfig structure contains the configuration information to configure the USB controller for DEVICE mode
XUsbPs_EndpointThe following data structure represents an endpoint used internally by the L0/L1 driver
XUsbPs_EpConfigEndpoint configuration structure
XUsbPs_EpInThe following data structure represents IN endpoint
XUsbPs_EpOutThe following data structures are used internally by the L0/L1 driver
XUsbPs_EpSetupData structures used to configure endpoints
XUsbPs_SetupDataThe following structure is used by the user to receive Setup Data from an endpoint
