u32 XDfeCcf_OverflowStatus::OverflowSwitch 

             In SWITCHABLE mode the channel in which

overflow first occurred.

This register is ignored in NON-SWITCHABLE mode.

  • 0 = DOWNLINK: Overflow occurred in the downlink channel.
  • 1 = UPLINK: Overflow occurred in the uplink channel.
