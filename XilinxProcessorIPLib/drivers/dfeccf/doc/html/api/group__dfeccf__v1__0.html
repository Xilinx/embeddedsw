XDfeCcf * XDfeCcf_InstanceInit ( const char *  DeviceNodeName)

API initialises one instance of a channel filter driver.

Traverses "/sys/bus/platform/device" directory (in Linux), to find registered CCF device with the name DeviceNodeName. The first available slot in the instances array XDfeCcf_ChFilter[] will be taken as a DeviceNodeName object.

Parameters
DeviceNodeNameis the device node name.
Returns
  • Pointer to the instance if successful.
  • NULL on error.
