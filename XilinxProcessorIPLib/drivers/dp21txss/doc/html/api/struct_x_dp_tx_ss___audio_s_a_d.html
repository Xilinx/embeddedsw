Detailed Description

Structure to store Short Audio Descriptor (SAD) information from CTA-861 extension blocks (3 bytes per SAD)

union { ... } XDpTxSs_AudioSAD::FormatDependent

Format-dependent data from SAD byte 2.

For LPCM (format code 1): Use BitDepthFlags for supported bit depths. For compressed formats (2-14): Use MaxBitrate for maximum bitrate in units of 8 kHz.
