Returns the SCL clock frequency currently configured in the instance. 


Parameters

  

    
InstancePtris a pointer to the XI3cPsx instance.

  

  




Returns
The SCL clock frequency in Hz as last successfully programmed by XI3cPsx_SetSClk() or XI3cPsx_SetSClkRate().


Note
This returns the software-stored rate, which is always kept in sync with the hardware. If XI3cPsx_SetSClkRate() fails, the previous valid rate is preserved and returned here. 



References XI3cPsx::IsReady, and XI3cPsx::SclkHz.











      

        

          s32 XI3cPsx_SetSClk 
          (

          InstancePtr)
          
        

      





Sets I3C Scl clock frequency. 


Parameters

  

    
InstancePtris a pointer to the XI3cPsx instance.

  

  




Returns
XST_SUCCESS if successful, XST_FAILURE if any computed timing count falls outside [SCL_I3C_TIMING_CNT_MIN, SCL_I3C_TIMING_CNT_MAX], indicating the input clock and SCL rate combination is not representable in the hardware timing registers.


Note
The SCL frequency is read from InstancePtr->SclkHz. To configure a custom SCL frequency, use XI3cPsx_SetSClkRate. 



References XI3cPsx_Config::BaseAddress, XI3cPsx::Config, XI3cPsx_Config::InputClockHz, XI3cPsx::IsReady, SCL_I3C_TIMING_CNT_MAX, XI3cPsx::SclkHz, XI3CPSX_SCL_I3C_OD_TIMING, XI3CPSX_SCL_I3C_PP_TIMING, and XI3cPsx_WriteReg.



Referenced by XI3cPsx_CfgInitialize(), and XI3cPsx_SetSClkRate().

Sets a custom I3C SCL clock frequency by updating the instance SCL rate and configuring the hardware registers accordingly. 


Parameters

  

    
InstancePtris a pointer to the XI3cPsx instance. 

    
SclkHzis the desired SCL clock frequency in Hz.

  

  




Returns
XST_SUCCESS if successful, XST_FAILURE if the requested SCL rate cannot be represented in the hardware timing registers for the given input clock.


Note
This updates InstancePtr->SclkHz and programs the hardware. For default SCL rate set during initialization, use XI3cPsx_SetSClk instead. 



References XI3cPsx::SclkHz, and XI3cPsx_SetSClk().
