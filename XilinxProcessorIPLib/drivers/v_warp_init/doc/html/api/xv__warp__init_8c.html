This function initializes the XV_warp_init driver instance using the provided configuration structure. Sets up base addresses and initializes the descriptor configuration.

Parameters
InstancePtrPointer to the XV_warp_init instance
ConfigPtrPointer to the configuration structure
Returns
XST_SUCCESS on successful initialization
Note
This function is only available in non-Linux builds. Both pointers must not be NULL.
