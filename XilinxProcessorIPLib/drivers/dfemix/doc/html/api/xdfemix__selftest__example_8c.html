xdfemix_selftest_example.c File Reference

Overview

This file contains a selftest example for using the Mixer hardware and Mixer driver.

This example does some writes to the hardware to do some sanity checks.

Note: MGT si570 oscillator is set to 152.25MHz by default. The DFE IP wrapper requires MGT clock to be set to 122.88MHz (some IP use 61.44MHz). Prerequisite is to set the MGT si570 oscillator to the required IP before running the example code. This is for the ZCU208 prodaction platform.
