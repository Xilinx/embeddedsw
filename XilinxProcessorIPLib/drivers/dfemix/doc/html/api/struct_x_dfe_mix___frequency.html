u32 XDfeMix_Frequency::TriggerUpdateFlag 

                Update the associated accumulator from

the FREQUENCY registers:

  • 0 = NO_UPDATE: Do not update
  • 1 = IMMEDIATE_UPDATE: Apply an immediate update
  • 2 = TRIGGER_UPDATE: Apply an update on the next CC_UPDATE trigger
