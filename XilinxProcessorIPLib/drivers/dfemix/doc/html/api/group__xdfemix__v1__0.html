XDfeMix * XDfeMix_InstanceInit ( const char *  DeviceNodeName)

API initialises one instance of a channel filter driver.

Traverses "/sys/bus/platform/device" directory (in Linux), to find registered MIX device with the name DeviceNodeName. The first available slot in the instances array XDfeMix_Mixer[] will be taken as a DeviceNodeName object.

Parameters
DeviceNodeNameis device node name.
Returns
  • Pointer to the instance if successful.
  • NULL on error.
