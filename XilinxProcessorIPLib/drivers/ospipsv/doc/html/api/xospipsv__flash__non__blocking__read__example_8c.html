xospipsv_flash_non_blocking_read_example.c File Reference

Overview

This file contains a design example using the OSPIPSV driver (xospipsv) The example writes to flash in IO mode and reads it back using non-blocking APIs in DMA mode.

It runs in polled mode. The hardware which this example runs on, must have an octal serial Flash (Micron) for it to run.

This example has been tested with the Micron Octal Serial Flash (mt35xu01gbba).

Note

None.
