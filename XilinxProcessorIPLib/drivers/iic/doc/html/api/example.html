Examples

You can refer to the below stated example applications for more details on how to use iic driver.

xiic_selftest_example.c

Contains an example on how to use the XIic driver directly. This example performs the basic selftest using the driver.

For details, see xiic_selftest_example.c.

xiic_dynamic_eeprom_example.c

Contains an example on how to use the XIic driver directly. This example consists of a Interrupt mode design which shows the usage of the Xilinx iic device and XIic driver to exercise the EEPROM in Dynamic controller mode. The XIic driver uses the complete FIFO functionality to transmit/receive data. This example writes/reads from the lower 256 bytes of the IIC EEPROMS.

For details, see xiic_dynamic_eeprom_example.c.
