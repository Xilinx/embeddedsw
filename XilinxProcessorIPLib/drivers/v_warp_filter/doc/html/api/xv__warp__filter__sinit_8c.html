s32 XV_warp_filter_Initialize ( XV_warp_filter *  InstancePtr,
UINTPTR  BaseAddress 
)

Initializes the Warp Filter instance using base address.

This function looks up the configuration for the specified base address and initializes the instance using the configuration found.

Parameters
InstancePtrPointer to the XV_warp_filter instance to initialize.
BaseAddressBase address of the Warp Filter control interface.
Returns
XST_SUCCESS if initialization is successful.
XST_DEVICE_NOT_FOUND if the base address is not found.
Note
This function is only available when SDT is defined.
InstancePtr must not be NULL.

References XV_warp_filter::IsReady, XV_warp_filter_CfgInitialize(), and XV_warp_filter_LookupConfig().

XV_warp_filter_Config* XV_warp_filter_LookupConfig ( UINTPTR  BaseAddress)

Looks up the device configuration based on the base address.

This function searches the configuration table for a device matching the specified base address and returns a pointer to its configuration structure. If BaseAddress is 0, returns the first configuration entry.

Parameters
BaseAddressBase address of the device to search for, or 0 for first entry.
Returns
Pointer to the configuration structure if found, NULL otherwise.
Note
This function is only available when SDT is defined.
