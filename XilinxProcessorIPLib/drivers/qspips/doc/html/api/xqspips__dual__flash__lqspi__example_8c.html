xqspips_dual_flash_lqspi_example.c File Reference

Overview

This file contains a design example using the QSPI driver (XQspiPs) in Linear QSPI mode, with two serial FLASH devices on seperate buses.

With two flash memories on seperate buses, even numbered bits in data words are written to the lower memory and odd numbered bits are written to the upper memory. This example writes to the two flash memories in QSPI mode and reads the data back from the flash memories, in Linear QSPI mode. It is recommended to use Manual CS + Auto start for best performance.

The hardware which this example runs on, must have a serial FLASH (Numonyx N25Q, Winbond W25Q, Spansion S25FL, ISSI IS25WP) for it to run. This example has been tested with the Numonyx Serial Flash (N25Q128) and IS25WP series flash parts.
