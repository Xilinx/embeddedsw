Enumeration Type Documentation

enum CamDeviceBufChainId_e

The buffer chain index enumeration of each output path.

Enumerator
CAMDEV_BUFCHAIN_MP 

ISP output main path buffer chain index.

CAMDEV_BUFCHAIN_SP1 

ISP output self1 path buffer chain index.

CAMDEV_BUFCHAIN_SP2 

ISP output self2 path buffer chain index.

CAMDEV_BUFCHAIN_RAW 

ISP output RAW path buffer chain index.

CAMDEV_BUFCHAIN_HDR_RAW 

ISP output retiming HDR RAW path buffer chain index.

CAMDEV_BUFCHAIN_METADATA 

Warning: CAMDEV_BUFCHAIN_METADATA will be deleted in next release, don't use this index.

CAMDEV_BUFCHAIN_RDMA 

ISP input read DMA path buffer chain index.

CAMDEV_BUFCHAIN_RETIMING 

ISP input retiming DMA path buffer chain index.

CAMDEV_BUFCHAIN_MAX 

Maximum path buffer chain index.

enum CamDeviceBufMode_e

The buffer mode enumeration which can be configured by user application.

Enumerator
CAMDEV_BUFMODE_INVALID 

Invalid buffer mode.

CAMDEV_BUFMODE_USERPTR 

The buffer allocated from user application.

CAMDEV_BUFMODE_RESMEM 

The buffer allocated from internal reserved memory.

CAMDEV_BUFMODE_MAX 

Maximum buffer mode.

enum CamDeviceBufQueBlockType_e

Enumerator

Non-blocking type.
