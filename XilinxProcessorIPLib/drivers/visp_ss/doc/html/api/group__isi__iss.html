#define ISI_BUSWIDTH_10BIT   ( ISI_BUSWIDTH_10BIT_EX )

Vin Interface Type.

#define ISI_BUSWIDTH_10BIT_EX   0x00000004

/formerly known as ISI_BUSWIDTH_10BIT (at times no marvin derivative was able to process more than 10 bit)

#define ISI_BUSWIDTH_8BIT_EX   0x00000002

to expand to a (possibly higher) resolution in marvin, the LSBs will be copied from the MSBs

#define ISI_BUSWIDTH_8BIT_ZZ   0x00000001

< BusWidth

to expand to a (possibly higher) resolution in marvin, the LSBs will be set to zero

#define ISI_INTERFACE_VERSION   6

interface version

please increment the version if you add something new to the interface.

This helps upper layer software to deal with different interface versions.

#define ISI_ITF_TYPE_BT1120   0x00000020

BT1120 conform data stream.

Mode, operating mode of the image sensor in terms of output data format and timing data transmission

BT601 conform data stream.

BT656 conform data stream.
