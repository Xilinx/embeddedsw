Example Applications for the driver visp_ss_v1_0

  • xvisp_ss_example.c (source) - Comprehensive VISP SS driver example demonstrating initialization, configuration, frame buffer writer setup, interrupt system, and memory management
  • xvisp_ss_example.h (source) - Header file containing function declarations and definitions for the VISP SS example
  • lscript.ld (source) - Linker script for memory layout configuration
  • test.cmake (source) - CMake configuration file for building the test examples with IO mode detection
  • image.h (source) - Raw image data in binary format for testing and validation
  • lilo.h (source) - LILO (Line-In/Line-Out) configuration JSON data in C array format
  • limo.h (source) - LIMO (Line-In/Memory-Out) configuration JSON data in C array format
  • mimo.h (source) - MIMO (Memory-In/Memory-Out) configuration JSON data in C array format
  • libvisp.a - VISP subsystem static library for standard configuration
  • libvisp_lilo.a - VISP subsystem static library optimized for LILO mode
  • reference_jsons/ - Reference JSON configuration files
    • lilo.json (source) - Line-In/Line-Out configuration reference
    • limo.json (source) - Line-In/Memory-Out configuration reference
    • mimo.json (source) - Memory-In/Memory-Out configuration reference
    • CMakeLists.txt (source) - Build configuration for reference JSON files

Description

The VISP SS (Video Image Signal Processing Subsystem) driver provides comprehensive support for real-time video processing applications including camera interfaces, video streaming, and image processing pipelines. The example applications demonstrate:

  • VISP SS core initialization and configuration
  • Frame buffer writer setup and management
  • Interrupt system configuration
  • Multiple I/O mode support (LILO, LIMO, MIMO)
  • JSON-based configuration management
  • Multiple output format support (RGB888, YUV420, YUV422)
  • Non-MCM (Multi-Channel Mode) operation
  • Memory management with aligned buffer allocation
  • Hardware reset and power management
