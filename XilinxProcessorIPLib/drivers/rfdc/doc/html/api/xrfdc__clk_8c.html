xrfdc_clk.c File Reference

Overview

This file contains a programming example for using the lmx2594 and lmk04028 clock generator.

This example configures the lmx2594 to 20 frequencies. If 0 is passed then the chip is powered down. For zcu111 board users are expected to define XPS_BOARD_ZCU111 macro while compiling this example.
