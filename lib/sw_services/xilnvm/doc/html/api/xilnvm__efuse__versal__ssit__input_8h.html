xilnvm_efuse_versal_ssit_input.h File Reference

Overview

This file contains macros which needs to configured by user based on the options selected by user operations will be performed on selected SLRs. Replace SLR_X with SLR_0, SLR_1, SLR_2 or SLR_3.

Note
   User configurable parameters for Versal eFUSE

    #define XNVM_EFUSE_SLR_X_ENABLE_PROGRAMMING             FALSE
    TRUE to enable programming of efuses in SLR_X
    FALSE will skip programming of all efuses in SLR_X

    #define XNVM_EFUSE_SLR_X_GLITCH_DET_WR_LOCK     FALSE
    TRUE permanently disables writing to ANLG_TRIM_3 row

Note: To program user eFuses you have to give the start User fuse number to be programmed at XNVM_EFUSE_SLR_X_PRGM_USER_FUSE_NUM and starting with that number of user fuses to be programmed should be mentioned at XNVM_EFUSE_SLR_X_NUM_OF_USER_FUSES. XNVM_EFUSE_SLR_X_USER_FUSES string should be multiples of 8 bytes always. Which means that if the string is 16 bytes, it will program 2 user eFuses. XNVM_EFUSE_SLR_X_PRGM_USER_FUSE_NUM range is 1 to 63.

Example : When XNVM_EFUSE_SLR_X_WRITE_USER_FUSES is TRUE and XNVM_EFUSE_SLR_X_PRGM_USER_FUSE_NUM is 4 - User eFuse number from where to start write. XNVM_EFUSE_SLR_X_NUM_OF_USER_FUSES is 2 - Number of User eFuses to program XNVM_EFUSE_SLR_X_USER_FUSES "1234578ABCDEF58" - User eFuse data to program UserFuse 4 will be programmed with 0x1234578 and UserFuse 5 will be programmed with 0xABCDEF58.

#define XNVM_EFUSE_SLR_X_READ_USER_FUSE_NUM XNVM_EFUSE_SLR_X_PRGM_USER_FUSE_NUM XNVM_EFUSE_SLR_X_READ_USER_FUSE_NUM - User eFuse number from where to start read. By default it reads from XNVM_EFUSE_SLR_X_PRGM_USER_FUSE_NUM

#define XNVM_EFUSE_SLR_X_READ_NUM_OF_USER_FUSES XNVM_EFUSE_SLR_X_NUM_OF_USER_FUSES XNVM_EFUSE_SLR_X_READ_NUM_OF_USER_FUSES - Number of eFuses to be read By default it reads XNVM_EFUSE_SLR_X_NUM_OF_USER_FUSES
