xilpuf_versal_net_example.c File Reference

Overview

This file illustrates encryption of red key, UDS and DME private keys using PUF KEY.

It can also be used for generating only PUF ID. To build this application, xilmailbox library must be included in BSP and xilsecure must be in client mode and xilpuf in server mode.
