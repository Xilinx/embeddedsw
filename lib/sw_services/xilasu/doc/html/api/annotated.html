Data Structures
Here are the data structures with brief descriptions:
Asu_AesParamsThis structure is common for both client and handler, which contains AES input and output parameters information
EcdsaCrvInfo< The structure below matches the more private structures of binary_curve and prime_curve in the SDK
XAsu_AesKeyObjectThis structure contains AES key object information
XAsu_ChannelMemoryThis struture is the channel's memory
XAsu_ChannelQueueThis structure is the channel's queue which is of 8 buffers
XAsu_ChannelQueueBufThis structure is the channel's queue buffer
XAsu_ClientParamsThis structure contains client parameters information
XAsu_CommChannelThis structure contains information of each communication channel
XAsu_CommChannelInfoThis structure contains information of all communication channels
XAsu_EccParamsThis structure contains ECC params info
XAsu_ReqBufThis structure is the request buffer
XAsu_RespBufThis structure is the response buffer
XAsu_RsaClientParamsThis structure contains RSA params info
XAsu_RsaCrtKeyCompThis structure contains RSA private key CRT parameters info
XAsu_RsaPubKeyCompThis structure contains RSA public key parameters info
XAsu_RsaPvtKeyCompThis structure contains RSA private key parameters info
XAsu_RsaRModNThis structure contains RSA exponent values (R mod N,R square mod N)
XAsu_RsaRRModNThis structure contains RSA exponent value (R square mod N)
XAsu_ShaOperationCmdThis structure contains SHA params info
