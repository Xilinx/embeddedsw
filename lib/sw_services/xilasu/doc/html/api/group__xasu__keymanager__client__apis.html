XAsu_KeyManagerParams *  KmSubVaultParamPtr 
)

This function sends a command to ASUFW to perform ECC key pair generation for the requested subsystem on the provided key size.

Parameters
ClientParamPtrPointer to the XAsu_ClientParams structure which holds the client input parameters.
KmSubVaultParamPtrPointer to XAsu_KeyManagerParams structure which holds the parameters of KeyVault input arguments.
Returns
  • XST_SUCCESS, if IPI request to ASU is sent successfully.
  • XASU_INVALID_ARGUMENT, if any argument is invalid.
  • XASU_QUEUE_FULL, if Queue buffer is full.
  • XST_FAILURE, if sending an IPI request to ASU fails.

Validation of inputs.

Validate the curve length and curve type for ECC.

Generate a unique ID and register the callback function. Two key IDs will be returned for ECC key pair generation command and stores at the address provided in KmSubVaultParamPtr->KeyIdAddr by the client:

  • First key ID will be the public key ID
  • Second key ID will be the private key ID
  • The size allocated for key IDs should be (XASU_KM_NUM_OF_KEY_IDS * XASU_KM_OUTPUT_ID_SIZE_IN_BYTES) to store both the key IDs.

Create command header.

Update request buffer and send an IPI request to ASU.
