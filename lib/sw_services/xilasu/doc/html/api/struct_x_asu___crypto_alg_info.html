Detailed Description

This structure contains data for cryptographic algorithms, including versioning and NIST compliance status, for up to maximum modules in ASUFW.

Field Documentation

u8 XAsu_CryptoAlgInfo::KatStatus

KAT status bit, 0 - KAT not run, 1 - KAT fail and 3 - KAT pass.

u8 XAsu_CryptoAlgInfo::NistStatus

NIST compliance status of the cryptographic algorithm.

0xFF: Algorithm is NIST compliant 0x00: Algorithm is not NIST compliant or NIST compliance is not applicable
