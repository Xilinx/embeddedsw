s32 XAsu_RsaValidateInputParams ( const XAsu_RsaParams *  RsaParamsPtr)

This function validates input parameters that are common to all RSA padding algorithms.

Parameters
RsaParamsPtrPointer to XAsu_RsaParams structure that holds the input parameters for RSA.
Returns
  • XST_SUCCESS, if input validation is successful.
  • XST_FAILURE, if input validation fails.

Validate that the addresses of all input buffers are non-zero.

References XAsu_RsaParams::InputDataAddr, and XAsu_RsaParams::KeyCompAddr.

Referenced by XAsu_RsaOaepDec(), XAsu_RsaOaepEnc(), XAsu_RsaPssSignGen(), and XAsu_RsaPssSignVer().
