s32 XAsu_ValidateKdfParameters ( const XAsu_KdfParams *  KdfParamsPtr)

This function validates the KDF input parameters.

Parameters
KdfParamsPtrPointer to XAsu_KdfParams structure which holds the parameters of KDF input arguments.
Returns
  • XST_SUCCESS, if KDF input parameters validation is successful.
  • XST_FAILURE, if KDF input parameters validation fails.

Validate SHA Mode and SHA Type.

Validate that exactly one of KeyInAddr or KeyId is provided.

When a direct key is provided, its length must be valid; context and output buffers must be valid.

References XAsu_KdfParams::ContextAddr, XAsu_KdfParams::ContextLen, XAsu_KdfHmacKeyObject::KeyId, XAsu_KdfHmacKeyObject::KeyInAddr, XAsu_KdfHmacKeyObject::KeyInLen, XAsu_KdfParams::KeyObject, XAsu_KdfParams::KeyOutAddr, XAsu_KdfParams::KeyOutLen, XAsu_KdfParams::ShaMode, XAsu_KdfParams::ShaType, XASU_KDF_MAX_CONTEXT_LEN, XASU_KDF_MAX_KEY_LENGTH, and XAsu_KmValidateKeyAddrNdKeyId().

Referenced by XAsu_HKdfGenerate(), and XAsu_KdfGenerate().
