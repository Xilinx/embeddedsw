s32 XAsu_OcpGetHuk ( XAsu_ClientParams *  ClientParamPtr,
u8 *  HukBuf,
u32  BufLen 
)

This function sends an IPI request to get the Hardware Unique Key.

Parameters
ClientParamPtris a pointer to XAsu_ClientParams which has parameters for the client request.
HukBufis a pointer to a buffer to hold the 256-bit Hardware Unique Key.
BufLenis the length of the buffer pointed to by HukBuf. It should not be less than XASU_OCP_HUK_SIZE_IN_BYTES (32 bytes).
Returns
  • XST_SUCCESS, if the function finishes successfully.
  • XASU_INVALID_ARGUMENT, if invalid parameters used.
  • XASU_INVALID_UNIQUE_ID, if failed to get unique ID.
  • XST_FAILURE, if sending an IPI request to ASU fails.

Validate input parameters.

Generate a unique ID and register the callback function.

Create command header.

Send an IPI request to ASUFW.
