Detailed Description

This structure contains info for key manager parameters.

Field Documentation

XAsu_AesKeyObject XAsu_KeyManagerParams::AesKeyObj

AES key object to be filled if input is wrapped key type.

This field is applicable only for Store Key API.

u64 XAsu_KeyManagerParams::KeyIdAddr

Address where generated key ID is stored.

For Store Key API: This field is mandatory. For Generate Key APIs: This field is optional and should be given only when the key to be stored in key vault.

Referenced by XAsu_KmGenerateAesIv(), XAsu_KmGenerateAesKey(), XAsu_KmGenerateEccKeyPair(), XAsu_KmGenerateRawKey(), XAsu_KmGenerateRsaKeyPair(), XAsu_KmStoreKey(), and XAsu_KmValidateVaultParams().

XAsu_KeyManagerKeyMetadata XAsu_KeyManagerParams::KeyMetadata

Key metadata.

Referenced by XAsu_KmGenerateAesIv(), XAsu_KmGenerateAesKey(), XAsu_KmGenerateEccKeyPair(), XAsu_KmGenerateRawKey(), XAsu_KmGenerateRsaKeyPair(), XAsu_KmStoreKey(), and XAsu_KmValidateVaultParams().

u64 XAsu_KeyManagerParams::KeyObjectAddr

Address of the key buffer.

For Store Key API: This field is mandatory and it is an input buffer. For Generate Key APIs: This field is optional and it is output buffer if the key needs to be returned to user.

Referenced by XAsu_KmStoreKey(), and XAsu_KmValidateVaultParams().

u8 XAsu_KeyManagerParams::Reserved[4]

Explicit padding to ensure consistent struct size across architectures.

u32 XAsu_KeyManagerParams::WrappedInputLen

Length of the wrapped key input data (applicable only for Store Key API).

This needs to be filled if input is wrapped key type.
