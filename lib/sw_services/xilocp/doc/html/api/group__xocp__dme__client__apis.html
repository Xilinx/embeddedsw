int XOcp_GenDmeResp ( XOcp_ClientInstance *  InstancePtr,
u64  NonceAddr,
u64  DmeStructResAddr 
)

This function sends IPI request to fill the DME structure and generates the response with signature.

Parameters
InstancePtr- Pointer to the client instance
NonceAddr- pointer to 48 bytes buffer which holds the Nonce, which shall be used to fill one of the member of DME structure
DmeStructResAddr- pointer to 224 bytes buffer, which is used to store the response DME structure of type XOcp_DmeResponseStructure
Returns
  • XST_SUCCESS - Upon success
  • XST_FAILURE - Upon any failure
  • Validate input parameters
  • Fill IPI payload for XOCP_API_GENDMERESP command and send the request to Server

References XOcp_ClientInstance::MailboxPtr, XOCP_ADDR_HIGH_SHIFT, XOCP_API_GENDMERESP, and XOcp_ProcessMailbox().
