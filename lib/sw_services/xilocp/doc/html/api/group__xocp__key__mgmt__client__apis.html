int XOcp_ClientAttestWithDevAk ( XOcp_ClientInstance *  InstancePtr,
u64  AttestWithDevAk 
)

This function sends IPI request to attest the data with DevAk.

Parameters
InstancePtr- Pointer to the client instance
AttestWithDevAk- Address of XOcp_AttestWithDevAk structure.
Returns
  • XST_SUCCESS - Attestation with DevAk is successful
  • XST_FAILURE - Upon any failure
  • Validate input parameters
  • Fill IPI payload for XOCP_API_ATTESTWITHDEVAK command and send the request to Server

int XOcp_ClientAttestWithKeyWrapDevAk ( XOcp_ClientInstance *  InstancePtr,
u64  AttnPloadAddr,
u32  AttnPloadSize,
u32  PubKeyOffset,
u64  SignatureAddr 
)

This function sends IPI request to calculate the hash of the Key Wrap buffer and attest with Key Wrap DevAk private key.

Parameters
InstancePtr- Pointer to the client instance
AttnPloadAddr- Address of the buffer which should be attested
AttnPloadSize- Size of buffer in bytes
PubKeyOffset- Offset in provided buffer where public key needs to be stored
SignatureAddr- Address of the signature after attestation
Returns
  • XST_SUCCESS - Successfully hashed and attested the data with Key Wrap DevAk private key
  • XST_FAILURE - Upon any failure
  • Validate input parameters
  • Fill IPI payload for XOCP_API_ATTEST_WITH_KEYWRAP_DEVAK command and send the request to Server

int XOcp_GenSharedSecretWithDevAk ( XOcp_ClientInstance *  InstancePtr,
const u8 *  PubKey,
u8 *  SharedSecret 
)

This function sends IPI request to generate shared secret using Elliptic Curve Diffie-Hellman Key Exchange (ECDH).

The private key used to generate the shared secret is internal DevAK private key which is determined by the subsystem from where the command is originating.

Parameters
InstancePtrPointer to the client instance
PubKeyPointer to the buffer which contains the public key to be used for calculating shared secret using ECDH.
SharedSecretPointer to the output buffer which shall be used to store shared secret
Returns
  • XST_SUCCESS On Success
  • Errorcode On failure
  • Validate input parameters
  • Fill IPI Payload for XOCP_API_GEN_SHARED_SECRET and send request to Server

References XOcp_ClientInstance::MailboxPtr, XOCP_ADDR_HIGH_SHIFT, XOCP_API_GEN_SHARED_SECRET, and XOcp_ProcessMailbox().
