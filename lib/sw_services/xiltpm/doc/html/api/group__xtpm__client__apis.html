int XTpm_GetPcrLog ( XTpm_ClientInstance *  InstancePtr,
u64  TpmPcrEventAddr,
u64  TpmPcrLogInfoAddr,
u32  NumOfLogEntries 
)

This function sends IPI request to get the log and status of PCR info from PLM.

Parameters
InstancePtr- Pointer to the client instance
TpmPcrEventAddr- Pointer to the PCR events structure
TpmPcrLogInfoAddr- Pointer to the PCR log info structure
NumOfLogEntries- Number of log entries to read
Returns
  • XST_SUCCESS - If PCR contents are copied
  • XST_FAILURE - Upon any failure

Send an IPI request to the PLM by using the CDO command to call TPM Get PCR log API and returns the status of the IPI response.

References XTPM_API_ID_GET_PCR_LOG, and XTpm_ProcessMailbox().
