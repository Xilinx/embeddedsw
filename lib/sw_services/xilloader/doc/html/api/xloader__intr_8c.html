This function is the interrupt handler for SBI data ready. 


In this handler, PDI is loadeed through SBI interface. SBI interface setting for JTAG/SMAP/AXI/HSDP should be set before this handler.


Parameters

  

    
DataNot used as of now, present as a part of general interrupt handler definition.

  

  




Returns
XST_SUCCESS on success and error code on failure
