int XLoader_AesKatTest ( XLoader_SecureParams *  SecurePtr)

This function performs KAT test on AES crypto Engine.

Parameters
SecurePtrPointer to the XLoader_SecureParams instance.
Returns
  • XST_SUCCESS on success
  • XST_FAILURE if setting AES data context lost fails
  • XLOADER_ERR_KAT_FAILED on AES DPACM or AES decrypt KAT failure
  • Update KAT status based on the user configuration
  • Set the data context of previous AES operation
  • Skip running the KAT for AES DPACM or AES if it is already run KAT will be run only when the CRYPTO_KAT_EN bits in eFUSE are set If KAT fails device will go into a secure lockdown state
  • Update KAT status in RTC area
  • Update KAT status based on the user configuration
  • Update KAT status in RTC area
