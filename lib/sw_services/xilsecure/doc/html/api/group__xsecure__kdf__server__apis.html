Function Documentation

int XSecure_Hkdf ( XSecure_KdfParams *  InDataPtr,
u8 *  KdfOut,
u32  KdfOutLen 
)

This function performs the HMAC based KDF.

Parameters
InDataPtris a pointer to the XSecure_KdfParams instance. one should initialize the structure with following parameters before calling this API.
  • Key - Is a pointer to the key
    • KeyLen - length of key pointer
    • Context - Is a pointer to the context for KDF
    • ContextLen - Specifies the length of the context pointer.
KdfOutis the pointer to output storage.
KdfOutLenis the size of output.
Returns
  • XST_SUCCESS - On success
  • XSECURE_ERR_HKDF_INVALID_PARAM - If input parameters are invalid
  • XST_FAILURE - On failure
  • Validate input parameters.
  • Calculate number of iterations based on HKDF output key length and SHA hash length.
  • Calculate HMAC and form KdfInput
  • Update block index of 1 byte to HMAC.
  • Update Context
  • Get final HMAC
  • HMAC output size is the SHA algorithm hash length. HKDF key output can be any length.
  • In the last iteration, data to be copied to the destination can be less than or equal to SHA algorithm hash length which varies based on the requested key output length.
  • Copy the final HMAC of each iteration to the destination key output buffer.
  • Verify loop index is within expected bounds
