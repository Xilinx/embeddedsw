int XSecure_MldsaSignGenerateClient ( XSecure_ClientInstance *  InstancePtr,
u64  MldsaParamAddr 
)

This function sends IPI request to generate MLDSA signature for a given data.

Parameters
InstancePtrPointer to the client instance
MldsaParamAddrAddress of XSecure_MldsaSignGenParams structure
Returns
  • XST_SUCCESS On success
  • XST_FAILURE On failure
  • Perform input parameter validation on InstancePtr. Return XST_FAILURE if input parameters are invalid
  • Fill IPI Payload
  • Send an IPI request to the PLM by using the CDO command to call XSecure_MldsaSignGenerate API and returns the status of the IPI response.

References XSECURE_ADDR_HIGH_SHIFT, XSECURE_API_MLDSA_SIGN_GENERATE, XSecure_ProcessMailbox(), and XSECURE_SLR_INDEX_SHIFT.
