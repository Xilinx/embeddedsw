XSecure_LmsPubKeyTmp_ Union Reference

Detailed Description

Used to store data which is repeatedly sent to SHA engine, during LMS root value calculation.

Size = 16(I) + 4(node_number/2) + 2(D_INTR) + (32 * 2)
