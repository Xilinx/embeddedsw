Detailed Description

This structure encapsulates all the parameters required to configure an instance of the SLH_DSA algorithm.

Field Documentation

XSecure_Slhdsa_a XSecure_SlhdsaParam::a

FORS: Height of each FORS tree (a): number of bits to select a leaf; t = 2^a leaves per tree.

Referenced by XSecure_SlhdsaForsPkFromSig().

XSecure_ChecksumParam XSecure_SlhdsaParam::ChecksumParams

WOTS+: Embedded checksum parameters for XSecure_ComputeChecksum used in WOTS+ within XMSS layers.

Referenced by XSecure_SlhdsaHtVerify(), XSecure_SlhdsaWotsPkFromSign(), and XSecure_SlhdsaXmssPkFromSign().

XSecure_Slhdsa_d XSecure_SlhdsaParam::d

HT: Number of XMSS layers (d): depth of hypertree, determines how many XMSS trees are stacked.

Referenced by XSecure_SlhdsaHtVerify().

XSecure_Slhdsa_h XSecure_SlhdsaParam::h

HT: Merkle tree height (h): total height of the hypertree combining all XMSS layers.

XSecure_Slhdsa_hprime XSecure_SlhdsaParam::hprime

XMSS: Subtree height (h'): height of each individual XMSS subtree, h' = h/d.

Referenced by XSecure_SlhdsaHtVerify(), and XSecure_SlhdsaXmssPkFromSign().

XSecure_Slhdsa_k XSecure_SlhdsaParam::k

FORS: Number of FORS trees (k): determines number of signature components from FORS layer.

Referenced by XSecure_SlhdsaForsPkFromSig().

XSecure_Slhdsa_lgw XSecure_SlhdsaParam::lgw

WOTS+: log2(Winternitz parameter) (lgw): lgw = log2(w), used for WOTS+ length calculations in XMSS layers.

XSecure_Slhdsa_m XSecure_SlhdsaParam::m

FORS: Message digest length (m): length of hash of message to be signed (in bits), typically >= k * a for FORS tree selection.

Referenced by XSecure_SlhdsaShake256sHashMsg().

XSecure_Slhdsa_n XSecure_SlhdsaParam::n

ALL: Output length parameter (n): core hash output and secret string size (in bytes).

Used by FORS, XMSS, WOTS+, and HT. Typical: 16, 24, 32.

Referenced by XSecure_SlhdsaForsPkFromSig(), XSecure_SlhdsaGetInstance(), XSecure_SlhdsaHtVerify(), XSecure_SlhdsaShake256sChain(), XSecure_SlhdsaShake256sHashF(), XSecure_SlhdsaShake256sHashH(), XSecure_SlhdsaShake256sHashMsg(), XSecure_SlhdsaShake256sHashTl(), XSecure_SlhdsaVerify(), XSecure_SlhdsaWotsPkFromSign(), and XSecure_SlhdsaXmssPkFromSign().

u32 XSecure_SlhdsaParam::NoOfInvSign

WOTS+: w - 1 = (2 ^ lgw) - 1, used in WOTS+ signature generation within XMSS trees.

Referenced by XSecure_SlhdsaWotsPkFromSign().

u32 XSecure_SlhdsaParam::SignLen

ALL: Total signature size in bytes (SignLen): (1 + k(1 + a) + h + d * len) * n.

Includes FORS signature, WOTS+ signatures, and HT authentication paths.
