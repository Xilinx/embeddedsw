int XSecure_LmsSignVerify ( XSecure_ClientInstance *  InstancePtr,
const XSecure_LmsParams *  ParamsPtr 
)

This function sends an IPI request to perform an LMS signature verification operation.

The data, signature and public-key buffers must be reachable by the server (i.e. placed in shared memory or DDR that the PLM can access) and their addresses / lengths must already be populated in the params structure. The caller is responsible for keeping ParamsPtr (and the buffers it references) alive and coherent until this call returns.

Parameters
InstancePtrPointer to the client instance.
ParamsPtrPointer to a populated XSecure_LmsParams structure.
Returns
  • XST_SUCCESS On successful signature verification.
  • XST_INVALID_PARAM If any input pointer is NULL.
  • XST_FAILURE If the verification or transport fails.

References XSECURE_ADDR_HIGH_SHIFT, XSECURE_API_LMS_SIGN_VERIFY, and XSecure_SendRequest().
