int XSecure_Sha2Kat ( XSecure_ClientInstance *  InstancePtr)

This function sends IPI request to PLM to perform SHA2 KAT.

Parameters
InstancePtrPointer to the client instance
Returns
  • XST_SUCCESS When KAT Pass
  • XST_INVALID_PARAM If input parameters are invalid
  • XST_FAILURE On failure
  • Fill IPI Payload
  • Send an IPI request to the PLM by using the CDO command to call XSecure_ShaKat API and returns the status of the IPI response.

References XSECURE_API_KAT, XSECURE_API_SHA2_KAT, XSecure_ProcessMailbox(), and XSECURE_SLR_INDEX_SHIFT.

Referenced by XSecure_KatPlatIpiHandler().

int XSecure_Sha2Operation ( XSecure_ClientInstance *  InstancePtr,
XSecure_ShaOpParams *  Sha2Params 
)

This function sends IPI request to generate SHA2 digest.

Parameters
InstancePtrPointer to the XSecure_ClientInstance structure.
Sha2ParamsPointer to the XSecure_ShaOpParams structure which holds parameters of SHA input arguments.
Returns
  • XST_SUCCESS, if IPI request to server is sent successfully.
  • XST_INVALID_PARAM, if any argument is invalid.
  • XST_FAILURE, if sending IPI request to server fails.
  • Perform input parameter validation on InstancePtr. Return XST_FAILURE if input parameters are invalid
  • Fill IPI Payload
  • Send an IPI request to the PLM by using the CDO command to call XSecure_ShaOperation api. Wait for IPI response from PLM with a timeout. If the timeout exceeds then error is returned otherwise it returns the status of the IPI response

References XSECURE_ADDR_HIGH_SHIFT, XSECURE_API_SHA2_OPERATION, XSECURE_DEBUG_GENERAL, XSecure_Printf, and XSecure_ProcessMailbox().
