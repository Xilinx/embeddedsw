int XSecure_Sha3Digest ( XSecure_Sha3 *  InstancePtr,
const UINTPTR  InDataAddr,
const u32  Size,
XSecure_Sha3Hash *  Sha3Hash 
)

This function calculates the SHA-3 digest on the given input data.

Parameters
InstancePtr- Pointer to the XSecure_Sha3 instance
InDataAddr- Starting address of the data on which sha3 hash should be calculated
Size- Size of the input data
Sha3Hash- Pointer to XSecure_Sha3Hash structure, where output hash is stored into Hash which is a member of XSecure_Sha3Hash structure
Returns
- XST_SUCCESS - If digest calculation done successfully
  • XSECURE_SHA3_INVALID_PARAM - On invalid parameter
  • XST_FAILURE - If any error from Sha3Update or Sha3Finish

References XSECURE_SHA3_INVALID_PARAM, XSecure_Sha3Finish(), XSecure_Sha3Start(), and XSecure_Sha3Update().

Referenced by XSecure_PpkVerify().

int XSecure_Sha3Finish ( XSecure_Sha3 *  InstancePtr,
XSecure_Sha3Hash *  Sha3Hash 
)

This function updates SHA3 engine with final data which includes SHA3 padding and reads final hash on complete data.

Parameters
InstancePtr- Pointer to the XSecure_Sha3 instance
Sha3Hash- Pointer to XSecure_Sha3Hash structure, where output hash is stored into Hash which is a member of XSecure_Sha3Hash structure
Returns
- XST_SUCCESS - If finished without any errors
  • XSECURE_SHA3_INVALID_PARAM - On invalid parameter
  • XSECURE_SHA3_STATE_MISMATCH_ERROR - If State mismatch is occurred
  • XST_FAILURE - If Sha3PadType is other than KECCAK or NIST

References XSecure_SetReset(), XSECURE_SHA3_INVALID_PARAM, XSECURE_SHA3_STATE_MISMATCH_ERROR, XSecure_Sha3ReadHash(), and XSecure_Sha3WaitForDone().

Referenced by XSecure_Sha3Digest(), XSecure_Sha3Hash(), XSecure_Sha3Kat(), and XSecure_SpkAuthentication().

int XSecure_Sha3Kat ( XSecure_Sha3 *  SecureSha3)

This function performs known answer test(KAT) on SHA crypto engine.

Parameters
SecureSha3- Pointer to the XSecure_Sha3 instance
Returns
  • XST_SUCCESS - When KAT Pass
  • XSECURE_SHA3_INVALID_PARAM - On invalid argument
  • XSECURE_SHA3_LAST_UPDATE_ERROR - Error when SHA3 last update fails
  • XSECURE_SHA3_KAT_FAILED_ERROR - Error when SHA3 hash not matched with expected hash
  • XSECURE_SHA3_PMC_DMA_UPDATE_ERROR - Error when DMA driver fails to update the data to SHA3
  • XSECURE_SHA3_FINISH_ERROR - Error when SHA3 finish fails

References XSecure_SetReset(), XSECURE_SHA3_FINISH_ERROR, XSECURE_SHA3_INVALID_PARAM, XSECURE_SHA3_KAT_FAILED_ERROR, XSECURE_SHA3_LAST_UPDATE_ERROR, XSECURE_SHA3_PMC_DMA_UPDATE_ERROR, XSecure_Sha3Finish(), XSecure_Sha3Hash(), XSecure_Sha3LastUpdate(), XSecure_Sha3Start(), and XSecure_Sha3Update().

int XSecure_Sha3LastUpdate ( XSecure_Sha3 *  InstancePtr)

This function notifies the SHA driver at the end of the SHA data update and last update includes padding also.

Typically called before XSecure_Sha3Finish to prevent driver adding the padding.

Parameters
InstancePtr- Pointer to the XSecure_Sha3 instance
Returns
- XST_SUCCESS - If last update can be accepted
  • XSECURE_SHA3_INVALID_PARAM - On invalid parameter
  • XSECURE_SHA3_STATE_MISMATCH_ERROR - If State mismatch is occurred
